## Installation
1. copy repo
    ```bash
    git clone --recursive https://github.com/daynlight/Shader_Editor.git
    ```
2. build via cmake
    ```bash
    mkdir -p build
    cd build
    cmake ..
    cmake --build . --config Release
    cd ..
    ```
3. install via cmake
    ```bash
      sudo cmake --install build/ --config Release
    ```
